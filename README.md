# 小红书仿站项目

## 项目简介
这是一个使用Vue3 + TypeScript + Element Plus开发的小红书网站仿站项目，主要用于学习和练习前端开发技术。

## 技术栈
- Vue 3
- TypeScript
- Element Plus
- Vite

## 功能特性
- 响应式布局设计
- 瀑布流展示
- 用户交互动画
- 分类导航
- 社交功能（点赞、评论）

## 本地开发

### 环境要求
- Node.js >= 16
- npm >= 8

```bash
npm install
```
### 启动开发服务器
```bash
npm run dev
```
